# UI 功能说明

本文档介绍 TRAE 论坛帖子展示页面的 UI 优化功能。

## 视图切换

页面提供两种视图模式：

### 分类视图（默认）
- 帖子按分类分组展示
- 每个分类显示为独立的列
- 便于快速浏览不同分类的帖子
- 点击帖子标题可在新标签页打开原帖

### 卡片视图
- 帖子以卡片形式展示
- 显示帖子封面图、标题、摘要和统计信息
- 支持按分类筛选后的单列展示
- 适合浏览帖子详细内容

## 排序功能

工具栏提供多种排序选项：

| 排序方式 | 说明 |
|---------|------|
| 最新发布 | 按发布时间倒序，最新的帖子在前 |
| 最早发布 | 按发布时间正序，最早的帖子在前 |
| 最多浏览 | 按浏览量排序，浏览量高的在前 |
| 最多点赞 | 按点赞数排序，点赞多的在前 |
| 最多回复 | 按回复数排序，回复多的在前 |

## 搜索功能

### 搜索范围
- 帖子标题
- 帖子摘要/内容
- 分类名称

### 搜索特性
- **实时搜索**：输入后 200ms 自动搜索
- **结果计数**：显示匹配的帖子数量
- **清除按钮**：点击 × 按钮一键清除搜索内容
- **分类联动**：搜索结果会更新分类标签的计数

## 骨架屏加载

页面加载时显示骨架屏，提供更好的用户体验：

- 减少页面加载时的视觉闪烁
- 平滑的内容过渡动画
- 提升感知加载速度

## 响应式设计

页面适配不同屏幕尺寸：

### 桌面端（>1200px）
- 完整的工具栏和视图切换
- 多列分类视图
- 卡片网格布局

### 平板端（768px-1200px）
- 隐藏视图切换按钮
- 自适应列数
- 优化触摸操作

### 移动端（<768px）
- 全宽搜索框
- 单列布局
- 简化的统计信息

## 键盘导航

页面支持键盘导航：

- **Tab 键**：在交互元素间切换
- **Enter 键**：激活按钮或链接
- **焦点指示器**：清晰的焦点样式

## 打印优化

页面包含打印样式：

- 隐藏工具栏和返回顶部按钮
- 静态头部，无阴影
- 分类列避免分页断裂

## 性能优化

- **懒加载图片**：卡片视图中的图片使用懒加载
- **CSS 变量**：统一的主题系统
- **平滑动画**：使用 CSS 过渡提升用户体验
- **防抖搜索**：减少不必要的搜索请求
