# 优化总结

本文档记录了项目从初始版本到当前版本的主要优化内容。

> 返回 [README.md](./README.md) 查看项目概览

---

## Python 爬虫脚本优化

### 1. 异步并发

- 使用 `aiohttp` 替代 `requests`，支持异步 HTTP 请求
- 添加并发控制（Semaphore），默认并发数 5
- 预计速度提升 5x（40s → 8s）

### 2. 增量更新

- 加载已有数据，只抓取新帖子
- 智能合并新旧数据，避免重复
- 减少 API 调用次数

### 3. 帖子详情刷新

- 每次运行强制刷新所有帖子的详情数据
- 确保标题、浏览量、点赞数等字段为最新值
- 使用 `asyncio.as_completed` 并发刷新，配合进度条显示

### 4. 日志系统

- 使用 `logging` 模块替代 `print`
- 支持日志级别、文件输出、格式化
- 日志文件保存在 `logs/` 目录

### 5. 数据模型

- 使用 Pydantic 定义数据模型
- 自动数据校验和类型检查
- 更好的 IDE 提示和代码可维护性

### 6. 数据质量校验

- 输出数据包含 `_quality` 质量报告
- 检测缺失标题、缺失日期、未知分类等异常
- 在日志中输出数据质量摘要

### 7. 代码重构

- 使用 `pathlib` 替代 `os.path`
- 使用 `html.unescape()` 替代手动替换
- 添加类型注解（Type Hints）
- 提取重复逻辑为公共方法
- 分类 ID 递归解析，支持子分类映射到父分类

## 前端优化

### 1. 代码分离

- CSS 分离为 `styles.css`
- JS 分离为 `app.js`
- 利于浏览器缓存和维护

### 2. 深色模式

- 使用 CSS 变量实现主题切换
- 支持系统主题自动检测
- 用户偏好保存到 localStorage
- URL 参数支持 `#theme=dark`

### 3. URL 状态同步

- 搜索词、分类、排序、视图模式同步到 URL hash
- 支持书签和分享
- 使用 `history.replaceState` 避免历史记录污染

### 4. 懒加载

- 卡片视图超过 100 个帖子时启用 IntersectionObserver 懒加载
- 仅渲染可视区域卡片，滚动到底部时批量加载
- 减少初始 DOM 节点数量

### 5. 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `/` | 聚焦搜索框 |
| `1` / `2` / `3` | 切换视图（分类/卡片/日历） |
| `D` | 切换主题 |
| `S` | 打开统计面板 |
| `R` | 刷新数据 |
| `Esc` | 清除搜索 / 关闭面板 |

### 6. 数据导出

- 支持导出为 CSV 格式（含 BOM 头，兼容中文）
- 支持导出为 JSON 格式
- 导出当前筛选后的数据

### 7. 统计图表

- 使用 Chart.js 绘制图表
- 分类分布环形图
- 发帖时间线柱状图
- 图表颜色与分类配置一致

### 8. 日历视图

- 月历网格展示帖子时间分布
- 分类色点标记帖子所属分类
- 点击日期展开帖子列表
- 月份前后导航

### 9. 自动刷新

- 页面内置 5 分钟间隔自动轮询
- 手动刷新按钮（带旋转动画）
- 页面切到后台暂停轮询，切回前台立即刷新

### 10. 代码重构

- SVG 图标提取为常量
- 使用模块模式封装应用状态
- 防抖函数优化搜索性能
- 统一的事件绑定和管理
- 数字格式化（万/千缩写）
- 相对时间显示（刚刚/N分钟前/N小时前等）

## GitHub Actions 优化

### 1. 并发控制

- 添加 `concurrency` 配置
- 避免多实例同时运行冲突

### 2. 错误处理

- 添加 `timeout-minutes` 限制
- 失败时上传日志文件
- 使用 `::error::` 输出 GitHub 错误提示

### 3. 构建摘要

- 提交信息包含帖子数量
- 验证步骤添加条件判断

### 4. 更新频率

- 从每 4 小时调整为每 2 小时
- 支持手动触发（`workflow_dispatch`）

## 性能对比

| 指标 | 优化前 | 优化后 |
|------|--------|--------|
| 爬取速度 | ~40s（同步） | ~8s（异步并发） |
| 数据新鲜度 | 全量抓取 | 强制刷新所有帖子详情 |
| 页面加载 | 内联 CSS/JS | 外部文件（可缓存） |
| 大量帖子 | 全量 DOM | IntersectionObserver 懒加载 |
| 主题切换 | 不支持 | 支持（CSS 变量 + localStorage） |
| 状态分享 | 不支持 | URL hash 同步 |
| 时间维度浏览 | 不支持 | 日历视图 |
| 数据导出 | 不支持 | CSV / JSON |
| 页面刷新 | 需手动 F5 | 自动轮询 + 刷新按钮 |
